ESTILOS CSS EXTERNO

La declaración externa de CSS, basada en declarar los estilos CSS en un archivo independiente que puede servir como referente para dotar de estilos a decenas, cientos o miles de archivos html, que únicamente deberán invocar el nombre de archivo utilizando una sintaxis específica. De este modo un cambio en los estilos habrá que hacerlo únicamente en el archivo de estilos correspondiente, lo cual supone una gran ventaja.
En el CSS Externo, al momento de guardar el archivo debe ser con un nombre y extensión css, por ejemplo estilos.css. Asegurarnos que la extensión del archivo sea css. No es válido si no tiene esta extensión.
La etiqueta link se usa en HTML para establecer vínculos entre un documento HTML y otros recursos que para nuestro caso es una hoja de estilos CSS, acompañada con la propiedad rel (relación o relationship) que el documento HTML debe usar los estilos definidos como texto/css en un archivo con el nombre indicado.
