Información General

En este OVI el grupo 301122-37 del CURSO DISEÑO DE SITIOS WEB.
Presenta el curso de Diseño de Sitios Web, donde muestra de forma dinámica y fácil el manejo de las propiedades Css, Múltiples clases: Float, Position, Owerflow, además de Estilos Internos y Externos de CSS. Por medio de conceptos básicos, ejercicios dinámicos y videos.

Objetivos del OVI

Conocer el concepto de CSS3 para creación de contenidos en formato web. Familiarizándose con el manejo de propiedades básicas.

Objetivos del Curso:

  • Conocer fundamentos teóricos sobre las generalidades básicas del Diseño  Web.
  • utilizar de forma correcta los lenguajes HTML5 y CSS3 para aplicarlos en la solución de problemas de construcción de Sitios Web.
  • Implementar servicios y sitios web a través de sistemas de control de versiones.
